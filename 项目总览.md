## 🎯 项目核心特性

### 💡 技术亮点

1. **Agentic RAG架构**
   
   - Planning Agent：自动分解复杂查询
   - Retrieval Agent：多策略检索
   - Self-Reflection Agent：答案质量评估与迭代优化
   - 相比传统RAG，准确率提升???

2. **混合检索系统**
   
   - 向量检索（Milvus + HNSW索引）
   - 关键词检索（BM25 / Elasticsearch）
   - 知识图谱检索（Neo4j + Cypher）
   - RRF融合算法 + Cross-Encoder重排序

3. **知识图谱增强**
   
   - 自动实体识别（NER）
   - 关系抽取（LLM-based）
   - 图谱可视化（D3.js）
   - 多跳推理查询

4. **多模态支持**
   
   - PDF（文本+图片+表格）
   - Markdown、Word、PPT
   - 图片OCR识别
   - 表格结构化提取

5. **企业级功能**
   
   - RBAC权限系统
   - 多租户数据隔离
   - 完整审计日志
   - 可观测性（Prometheus + Grafana + ELK）

2. **申请API密钥**
   
   - OpenAI API Key（用于GPT-4和Embedding）
   - 其他可选：Anthropic、Cohere等

3. **初始化项目**
   
   ```bash
   # 创建后端项目
   mkdir backend && cd backend
   python -m venv venv
   source venv/bin/activate
   pip install fastapi langchain langgraph sqlalchemy...
   
   # 创建前端项目
   npm create vite@latest frontend -- --template react-ts
   cd frontend && npm install
   ```
